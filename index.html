Godot Module Template Maker

Project started approximately 4/26/2022. This project is written as a C++ console application, and its main purpose is to assist with writing the boilerplate code for custom modules for the Godot Engine.

Some notable functions include checking whether or not a file or directory exists, backing up modules, and creating or deleting files from the disk.

Blender Render Farm

Project started around 6/19/2021. This project involved configuring hardware and python scripts on a total of 20 nodes of Odroid-MC1 "My Cluster One" (5 units, 4 nodes each). Each unit has 32 CPU cores and 8 GB of DRAM, so the total cluster has 160 CPU cores and 40 GB DRAM. The units are mounted to a small table, underneath which is a 24 port Gig-speed switch and 2x 5V 60A Power Supply units to power the cluster.

The node configuration was done via ssh and Linux terminal commands, as the nodes do not include any display output port, only a UART connection for debugging purposes. I later also added a Raspberry Pi Zero-W to control 2 LED panels to display status messages (not pictured).

The purpose of the cluster, known as the "Tiny Potato Collective" due to being both physically small and not super powerful, is to offload the work of rendering 3D images created with Blender from my main PC. Rendering images and animations can take a long time, so this way I'm free to work on other things while the cluster works in the background.

The Python script runs on another machine and delegates the next frame to be rendered to the next available machine so that the load is balanced as evenly as possible across all nodes for maximum efficiency.

Windows Police Pro Malware Remover

This was a project which I made out of necessity around September 2009. My wife at the time had a PC which became infected with the Windows Police Pro malware.

The malware was fairly new and caused a lot of problems, including making normal executable ".exe" files unable to run as normal. I wrote a few Windows batch file scripts and registry files ".reg", as well as some VBScript commands to help restore the system. There was a command line version as well as later a Windows GUI version of the program available. I also hosted a website which had several updates as I added new features and learned more about how to remove this malware.

The website is not currently being hosted anymore as by now, most major antivirus programs easily detect and remove this malware, but at the time my program was downloaded a few hundred times and I even received a few donations from some people I was able to help to remove the malware from their systems.
